# Property Management System (PMS)

宿泊施設の運営を効率化し、収益最大化を支援する統合管理システム。

## モジュール概要

本システムは以下の7つのモジュールで構成されています。

### 1. [施設管理](specs/1_FacilityManagement/README.md)
施設・部屋タイプ・部屋の基本情報を一元管理します。
- 複数施設の管理
- 部屋タイプごとの特性・料金設定
- 部屋の状態ステータス管理（Vacant, Occupied, Dirty, etc.）

### 2. [人事管理](specs/2_StaffManagement/README.md)
スタッフ情報と権限を管理し、適切なアクセス制御を実現します。
- スタッフ情報の登録・更新
- 役割ベースの権限管理（Admin, Front Desk, Housekeeping, etc.）
- パスワード管理

### 3. [料金管理](specs/3_PricingManagement/README.md)
シーズンやOTAごとの柔軟な料金設定と、サイトコントローラー連携を担います。
- シーズン別・OTA別のレートプラン設定
- サイトコントローラーとの自動同期
- ダイナミックプライシング対応

### 4. [予約管理](specs/4_ReservationManagement/README.md)
予約の取り込みから部屋アサインまでを一元管理し、ダブルブッキングを防止します。
- サイトコントローラーからの予約自動取り込み
- ダブルブッキング検出と調整
- 予約カレンダーとアサイン機能

### 5. [宿泊者管理](specs/5_GuestManagement/README.md)
チェックイン・チェックアウト処理と、法令に基づく宿泊者名簿の管理を行います。
- 事前案内メールとデジタルトークンによる名簿収集
- スムーズなチェックイン・チェックアウト処理
- 宿泊者名簿の保存・検索

### 6. [清掃管理](specs/6_HousekeepingManagement/README.md)
チェックアウト後の部屋清掃を効率化し、モバイル端末でのリアルタイム連携を実現します。
- 清掃優先順位の自動算出（チェックイン時刻ベース）
- モバイル端末による状態更新
- 備品不足の記録と補充依頼

### 7. [売上分析](specs/7_SalesAnalysis/README.md)
詳細な売上データの集計と分析により、経営判断を支援します。
- ADR, OCC, RevPARなどの自動算出
- OTA別・属性別の多角的な売上分析
- 監査対応用のデータ保存

## 主な機能特徴

- **サイトコントローラー連携**: 予約・料金・在庫の自動同期
- **モバイル対応**: 清掃スタッフ向けのモバイル最適化画面
- **デジタル宿泊者名簿**: 事前メールとWebフォームによる効率的な名簿収集
- **権限管理**: スタッフの役割に応じたきめ細やかなアクセス制御
- **分析レポート**: 経営指標の可視化とレポーティング
