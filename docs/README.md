# ホテル施設管理システム（PMS Admin）

## 概要
ホテル運営会社向けの総合施設管理システム。
客室・設備の管理から、スタッフ配置、料金設定、予約管理、ゲスト対応、清掃管理、売上分析まで、
ホテル運営に必要なすべての業務を一元的に管理・運用できる。

## 主な機能
1. **施設管理** - 施設・部屋タイプ・部屋の登録・編集、部屋状態管理、メンテナンス設定
2. **スタッフ管理** - スタッフ情報管理、アカウント発行、権限ロール設定、パスワードリセット
3. **料金管理** - シーズン設定、レートプラン管理、OTA連携、サイトコントローラー同期
4. **予約管理** - OTA予約取り込み、予約一覧・カレンダー、部屋アサイン、ダブルブッキング対応、キャンセル処理
5. **宿泊者管理** - チェックイン案内メール、宿泊者名簿管理、チェックイン・チェックアウト処理
6. **清掃管理** - 清掃対象一覧、清掃状態管理、備品不足報告、清掃完了通知
7. **売上分析** - 売上集計、KPI分析（ADR・OCC・RevPAR）、OTA別分析、宿泊者属性分析

## 対象ユーザー
- ホテル管理者（施設オーナー/総支配人）
- 人事担当者
- フロントスタッフ
- 予約管理担当者
- 清掃スタッフ
- 経理担当者

## ナビゲーション構成
- ダッシュボード
- 施設管理（施設検索、部屋タイプ管理、部屋管理）
- スタッフ管理（スタッフ一覧、権限管理）
- 料金管理（シーズン管理、レートプラン管理、同期ログ）
- 予約管理（予約ダッシュボード、予約一覧、予約カレンダー、部屋アサイン、ダブルブッキング対応）
- 宿泊者管理（到着予定一覧、チェックイン処理、チェックアウト処理、宿泊者名簿検索、案内メール送信）
- 清掃管理（清掃ダッシュボード、清掃対象一覧、部屋状況一覧）
- 売上分析（売上ダッシュボード、売上集計、分析指標、OTA別分析、宿泊者属性分析）
