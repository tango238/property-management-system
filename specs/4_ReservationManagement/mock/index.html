予約管理 - モック画面一覧

サイトコントローラーからのOTA予約取り込み、ダブルブッキングの検出・調整、予約の確認・更新、部屋のアサイン、キャンセル処理を提供

メイン画面フロー

予約管理ダッシュボード → 予約取り込み画面 → 予約一覧画面 → 予約詳細画面 / 部屋アサイン画面 / 予約キャンセル画面
ダブルブッキング検出時: ダブルブッキング一覧画面 → 再割り当て画面
予約検索: 予約検索画面 → 予約詳細画面

ダッシュボード

  • 予約管理ダッシュボード
    予約管理の全体状況を一覧で確認し、各種操作への導線を提供

予約取り込み

  • 予約取り込み画面
    サイトコントローラーからOTA予約を取り込み、取り込み状況を確認
  • 取り込みエラー一覧画面
    予約取り込み時に発生したエラーを確認し、対応

ダブルブッキング管理

  • ダブルブッキング一覧画面
    ダブルブッキングの状況を確認し、部屋の再割り当てを行う
  • ダブルブッキング再割り当て画面
    ダブルブッキング発生時に別の部屋を選択して再割り当て

予約一覧・カレンダー・検索

  • 予約一覧画面
    予約情報をリスト形式で確認し、各種操作を行う
  • 予約カレンダー画面
    予約状況をカレンダー形式で視覚的に確認
  • 予約検索画面
    条件を指定して予約を検索

予約詳細・更新

  • 予約詳細画面
    予約の詳細情報を確認し、変更やゲスト要望の記録を行う

部屋アサイン

  • 部屋アサイン画面
    予約に対して具体的な部屋番号を割り当てる

予約キャンセル

  • 予約キャンセル画面
    予約をキャンセルし、キャンセル料を判定
