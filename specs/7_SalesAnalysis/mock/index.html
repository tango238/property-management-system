売上分析 - モック画面一覧

宿泊施設の経営判断に必要な売上データの集計・分析を行う機能のモック画面です。ADR・OCC・RevPARなどの分析指標を算出し、OTA別・宿泊者属性別の売上分析を通じて、収益最大化の戦略立案を支援します。

メイン画面フロー

ダッシュボード → 売上集計 / 分析指標 / OTA別分析 / 属性別分析 → 各詳細画面 → レポート出力

ダッシュボード

  • 売上分析ダッシュボード
    日々の稼働状況とKPIを一目で把握するためのホーム画面

売上レポート作成

  • 売上集計画面
    集計期間を指定して売上金額を集計しレポート形式で出力する
  • 売上データ検索画面
    過去の売上データを検索しエクスポートする（監査対応用）

分析指標

  • 分析指標画面
    ADR・OCC・RevPARを確認し経営判断の基礎データを得る
  • 月次KPI分析画面
    月次KPIを前月・前年同月と比較し業績評価を行う

売上推移分析

  • 売上推移分析画面
    年間の売上トレンドと季節変動パターンを分析する
  • 売上減少分析画面
    売上減少月の要因を分析し改善施策を立案する

OTA別分析

  • OTA別売上分析画面
    OTA別に売上・手数料・純利益を分析しチャネル戦略を検討する
  • 直接予約分析画面
    直接予約比率を分析し手数料負担削減施策を検討する

宿泊者属性別分析

  • 宿泊者属性分析画面
    宿泊者属性別に売上を分析しターゲット顧客層を特定する
  • インバウンド分析画面
    国籍別売上を分析しインバウンド戦略を立案する
  • 利用目的分析画面
    利用目的別の平均単価を分析し高単価セグメントを特定する

総合分析

  • 総合分析画面
    売上集計からKPI・OTA・属性分析まで一連のレポートを作成する
