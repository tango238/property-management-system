# 画面遷移図 - 売上分析

## 概要

売上分析機能は、宿泊施設の経営判断に必要な売上データの集計・分析を行う機能です。経理担当者がADR・OCC・RevPARなどの分析指標を算出し、OTA別・宿泊者属性別の売上分析を通じて、収益最大化の戦略立案を支援します。

## ユーザーロール

- **経理担当者**: 売上レポートの作成、分析指標の確認、OTA別・属性別分析、監査対応を担当する責任者

## 重要な遷移

### 売上集計からレポート出力まで

- **起点**: 売上集計画面
- **終点**: レポート出力完了
- **トリガー**: 集計実行ボタンクリック
- **条件**: 集計期間が正しく設定されている（開始日＜終了日）
- **関連ユースケース**: 月次売上レポートの作成と出力
- **備考**: OTA手数料は自動計算される

### KPI比較から改善ポイント特定まで

- **起点**: 月次KPI分析画面
- **終点**: 改善ポイント特定
- **トリガー**: 比較実行ボタンクリック
- **条件**: 対象月のデータが集計済み
- **関連ユースケース**: 月次KPI分析と比較
- **備考**: 目標値を設定すると達成率も表示される

### 売上減少月の要因分析

- **起点**: 売上推移分析画面
- **終点**: 売上減少分析画面
- **トリガー**: 減少月をクリック
- **条件**: 前年同月比で減少している月が存在
- **関連ユースケース**: 売上減少要因の分析
- **備考**: 稼働率と平均単価の変動を分解して分析

### OTA分析から直接予約施策検討まで

- **起点**: OTA別売上分析画面
- **終点**: 直接予約分析画面
- **トリガー**: 直接予約分析へ遷移
- **条件**: OTA経由の予約比率が高い
- **関連ユースケース**: 直接予約比率向上施策の検討
- **備考**: 手数料削減シミュレーションが可能

```

└─→ データ不足: 警告メッセージ表示

```

## 備考

- すべての分析画面でPDF/Excel形式のレポート出力が可能
- 売上データは監査対応のため7年間保存される
- 分析指標（ADR・OCC・RevPAR）は売上集計時に自動更新される
- OTA手数料はOTA種別に応じた手数料率が自動適用される
- OTA種別: 楽天トラベル、じゃらん、Booking.com、Airbnb、直接予約
